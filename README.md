# Ebook Manager

Aplicación en Python + Streamlit para gestionar una biblioteca de ebooks.

## Características

- Escaneo automático de carpetas
- Soporte EPUB
- Extracción de portadas
- Filtros por autor
- Búsqueda por título
- Apertura directa de ebooks
- Cache local JSON

## Tecnologías

- Python
- Streamlit
- EbookLib
- Pillow

## Ejecutar proyecto

```bash
streamlit run app.py
